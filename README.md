## Project clone

```
git clone https://github.com/dejota/challenge-pokeapi.git

cd challenge-pokeapi
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### View on browser

```
http://localhost:5173
```

### Compiles and minifies for production

```
npm run build
```

### Run tests

```
npm run test
```
